# Contributing to Sentinel-Shield

Thank you for your interest in contributing to Sentinel-Shield.

## Getting Started

1. Fork the repository.
2. Create a feature branch from `main`.
3. Install dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
4. Make your changes and add tests where applicable.
5. Submit a pull request.

## Code Style

- Follow PEP 8 conventions.
- Use type hints for all function signatures.
- Keep functions small and focused.

## Reporting Issues

Open an issue on GitHub with a clear description, steps to reproduce, and expected vs. actual behavior.
